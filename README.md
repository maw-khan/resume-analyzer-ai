# 📄 Resume Analyzer AI

AI-powered ATS resume analyzer that compares resumes with job descriptions and provides match scoring and improvement suggestions.

---

## 🚀 Features
- Resume PDF parsing
- Job description analysis
- ATS match scoring
- Skill gap detection
- AI-powered feedback using Gemini
- Clean Streamlit UI

---

## 🧠 Tech Stack
- Python
- Streamlit
- Gemini API (Google GenAI)
- NLP (keyword extraction)

---

## 🏗️ Architecture
Resume → Text Extraction → Keyword Analysis → Scoring → LLM Feedback → UI

---

## 📦 Installation

```bash
git clone <repo>
cd resume-analyzer-ai
pip install -r requirements.txt
streamlit run app.py
```
